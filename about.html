Lore

Cobblecade was created by my friends and I for fun, just so we can do something on the side, because we do nothing all day long...

We like playing minigames and we connect a lot when we play minecraft so we thought it was a good idea to open a server up so that others can experience the fun that we get from playing these minigames!

We needed a family-friendly name so we stuck with Cobblecade over other options...
